Tech Stack Recommendation Document
Ajinkya Marine Pvt Ltd Website
1. Frontend Framework Recommendation
Recommended: Next.js (React Framework)
Next.js is recommended due to strong SEO support (SSR & SSG), high performance, image optimization, built-in routing, and production readiness for B2B websites.
2. Styling Solution
Recommended: Tailwind CSS
Tailwind enables rapid UI development, consistent design systems, smaller CSS bundles, and precise control over styling for modern industrial design.
3. Backend Technology
Recommended: Node.js (Next.js API Routes or Express)
Node.js allows seamless integration with the frontend, handles RFQ forms, file uploads, authentication, and scales efficiently for expected traffic.
4. Database Recommendation
Recommended: PostgreSQL
PostgreSQL provides structured relational data handling, scalability, security, and is ideal for managing products, inquiries, RFQs, and admin users.
5. Essential Libraries & Packages
Frontend: Tailwind CSS, React Hook Form, Zod, Axios, Framer Motion
Backend: Prisma ORM, Multer, Nodemailer, JWT Authentication
SEO: next-seo, next-sitemap
Analytics: Google Analytics 4, Google Tag Manager
6. Deployment & Hosting
Frontend & Backend Deployment: Vercel
Database Hosting: Supabase or Neon (Managed PostgreSQL)
File Storage: AWS S3 or Supabase Storage
This setup ensures fast global CDN delivery, automatic CI/CD, scalability, and secure storage.
7. Development Tools & Workflow
Development Tools: VS Code, GitHub, Postman, Figma
Workflow: Design in Figma ? Build frontend ? Create backend APIs ? Connect database ? Integrate forms ? Optimize SEO ? Deploy ? Performance audit.
Performance & Scaling Strategy
Static generation for product pages, image optimization, lazy loading, CDN usage, and asset minification ensure load times under 2 seconds.
The stack supports scaling beyond 10,000 monthly visitors and allows future mobile app expansion.
